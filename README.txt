- shared_functions.R stores most of the functions that are used to compile the reports
- compile_templates.R is used to create all of the reports
- Results stores all the reports and other stuff related to the reports
- Templates stores all the templates used to create the reports
- All statewide mean metrics are calculating using a weighted average 
of the countywide metrics with the number of households in the counties as the weights
