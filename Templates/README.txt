- TemplateLong.Rnw a template for long states
- TemplateWide.Rnw is a template for wide states
- TemplateDC.Rnw is a template for DC (no maps)
- Template.Rnw is a template for all the other states with data
- Hawaii and Mississippi have no data, so reports for them are not created
- State type is defined using the aspect ratio for that state's map
	- Regular corresponds to an aspect ratio betweem 0.8 and 1.25
	- Wide corresponds to an aspect ratio greater than 1.25
	- Long corresponds to an aspect ratio less than 0.8
